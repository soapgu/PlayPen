## 前言

开卷有益。

写在最前，长期以来，技术学习并没有养成分享的习惯。没有好好成体系的整理自己所学的知识，时间长了也容易遗忘。

仓库名起名**PlayPen**，这个名字源于我上一家公司一个老外同事（名字已经不记得了），他把相关的代码Demo放置在这个名字下面。我在当时经常到这里面去到扒好东西，从中受益匪浅。起这个名字的初衷也希望我能帮助到别人。
